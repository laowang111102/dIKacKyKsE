# 行人目标检测系统 python752

## 项目概述

本项目是一款基于Python语言的深度学习目标检测系统，专注于行人检测。系统使用YOLOv8作为预训练模型，旨在实现对行人目标的快速准确识别。

## 技术栈

- **后端**：Python，YOLOv8
- **前端界面**：PyQt5

## 功能模块

- **行人检测**：使用训练好的模型进行实时行人目标检测
- **模型训练**：支持用户自定义数据集进行模型训练
- **模型部署**：将训练好的模型集成到前端界面中，实现用户交互

## 系统角色

- **用户**：使用前端界面进行行人检测操作
- **开发者**：可训练和调整模型，维护系统运行

## 运行环境

- Python环境（含必要库：YOLOv8，PyQt5等）

## 部署步骤

1. 准备Python运行环境，安装必要的库
2. 下载或训练模型，放置在指定路径
3. 运行前端界面代码`qt.py`，启动系统

## 目录结构

```
.
├── runs
│   └── train
│       └── exp
│           └── weights
│               └── best.pt  (训练好的模型)
├── test_0.jpg
├── test_1.jpg
├── train_model.py  (模型训练脚本)
└── qt.py           (前端界面代码)
```

## 核心亮点

- **实时检测**：基于YOLOv8的高效算法，实现实时行人目标检测
- **用户友好**：简洁的前端界面，便于用户操作
- **自定义训练**：允许用户使用自己的数据集进行模型训练，提高识别准确性

**注意**：由于数据集限制，模型准确率可能不完全满足所有场景需求。在使用过程中，请遵守相关法律法规，确保技术应用的合规性。
